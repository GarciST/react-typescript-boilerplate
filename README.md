## How to start
 - Clone project
```
git clone https://github.com/GarciST/react-typescript-boilerplate.git
```

 - Intall dependecies
 ```
 npm install
 ```

  - Run tests
 ```
 npm test
 ```

  - Run application
 ```
 npm start
 ```
